**Event Detection in Bangla: Evaluating Language Model Robustness to Noisy Text**

- This repository contains the training and evaluation pipeline for assessing the robustness of Large Language Models & PLMS (BERT-based) [A work-in-progress paper]

- A sample structure is provided in `/sample_data` 
- Training codes are in `/training`
- COde to generate synthetic noise mimicking  Bangla Typing behaviour in `/noise_generator` (based on Sifat etal, 2020)
